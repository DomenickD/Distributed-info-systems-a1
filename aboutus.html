About Me

Hello! I'm Domenick Dobbs, a graduate student at the University of South Florida pursuing a Master's in AI. With a background in web development, Data Science, and AI programming; I specialize in building responsive and accessible tools for customers. This project showcases my lesser used skills in HTML, CSS, and JavaScript, along with my ability to implement CRUD functionality and data visualization using Chart.js. I'm excited to share my work with you!

My Approach

I start with building out the structure in html. Then I add styles using CSS custom properties which I often find inspiration from online. Finally I add interactivity with vanilla JavaScript. The goal is to keep things simple, clean, and easy to build on if need be.

  • Clean, readable code
  • Consistent design tokens
  • Minimal dependencies

Tech Stack

  • HTML5 for structure
  • CSS with custom properties for a modern theme
  • JavaScript (vanilla) for simple interactions
  • Chart.js on the Analytics page (via CDN)
  • D3.js on the Analytics page (via CDN)
